# 💬 Real-Time Chat Application

A full-stack real-time chat application built using the MERN stack and Socket.IO. Users can chat globally, send private messages, manage friends, and see online/offline status in real time.

## 🚀 Live Demo

LINK : https://chat-app-pi-plum-26.vercel.app


## ✨ Features

- User Authentication (Register/Login)
- Email Verification
- Global Chat Room
- Private One-to-One Chat
- Friend Requests
- Accept/Reject Friend Requests
- Remove Friends
- Real-Time Messaging using Socket.IO
- Online/Offline User Status
- Unread Message Notifications
- Responsive UI
- Secure JWT Authentication

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM
- Socket.IO Client
- React Toastify

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Socket.IO
- JWT Authentication
- Bcrypt

### Deployment
- Vercel (Frontend)
- Render (Backend)
- MongoDB Atlas (Database)
